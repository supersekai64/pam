# Getting Started

## Prerequisites

- Node.js >= 20.0.0
- PNPM >= 9.0.0

## Installation

```bash
pnpm install
```

## Build

```bash
pnpm build
```

## Usage

```bash
memory init global
memory init project
memory add
memory list
memory search "query"
```

## Development

```bash
pnpm test
pnpm lint
pnpm format
```
